3. **DISTILL** — compress to the replicable essence (not the story — the algorithm)
4. **ENCODE** — write as skill.md (input → process → output → rubric score)

## TRANSMUTATION FORMULA

```
RAW DATA IN

  ↓ extract: the verb (what ACTION does this enable?)
  ↓ abstract: make it domain-independent
  ↓ encode: input → transformation → output → feedback
  ↓ score: rubric (leverage/scalability/reusability/speed/monetization/automation)
  ↓ if score >= 7.0: create skill.md
  ↓ if score >= 8.5: flag as PRIORITY — compound immediately
OUTPUT: self-contained skill.md
```
