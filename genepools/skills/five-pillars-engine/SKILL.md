## FRAMEWORK 5: BRAINSTORMING ENGINE SUITE (5 Modes)

### Mode A — AI-Augmented Ideation
```
1. Prompt Precision   → Multi-layered AI prompts (context + constraints + role + examples)
2. Quantity Surge     → 50+ ideas fast, volume > perfection
3. Human-AI Mash-Up  → Remix AI outputs with intuition: "what if" fusions
4. Pause & Recharge   → 5-30 min break → return with fresh pattern recognition
5. Rapid Convergence  → Cluster → vote → prototype top 3-5 in <20 min
```

### Mode B — Lightning Decision Jam (LDJ Speed)
```
1. Silent Sprint      → Individual generation, no talking, 7-10 ideas in 10 min
2. Question Reframe   → Generate 20+ questions BEFORE answers → kill assumptions
3. Worst-to-Best Flip → How would this fail? → reverse for breakthrough safeguards
4. Rapid Mash-Up      → Force random 3-element combinations → repeat 2-3x
5. Consensus Sprint   → Dot-vote → assign owners → move in <15 min
```

### Mode C — Question-First Innovation Engine
```
1. Problem Explosion  → 30+ questions, zero answers (reveals blind spots)
2. Assumption Hunt    → Identify + flip hidden assumptions for new angles
3. Starburst Reframe  → Who/what/where/when/why/how each question
4. Cross-Domain Fusion → Apply questions from unrelated fields (biology→business)
5. Solution Synthesis → Questions become idea seeds → prototype most provocative
```
